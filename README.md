# Film Reviewer

Film Reviewer is a React app for uploading and reviewing MP4 videos in the browser.

## What it does

- Provides a simple sign-in gate for accessing the gallery.
- Lets users upload MP4 videos and stores them locally in IndexedDB.
- Generates and saves video thumbnails for quick previews.
- Supports editing video titles, opening videos in a player, and deleting saved items.

## Tech stack

- React + TypeScript
- Vite
- React Router
- Bootstrap + Tailwind CSS

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal (usually `http://localhost:5173`).

## Available scripts

- `npm run dev` - start the development server
- `npm run build` - create a production build
- `npm run preview` - preview the production build locally
- `npm run lint` - run ESLint
